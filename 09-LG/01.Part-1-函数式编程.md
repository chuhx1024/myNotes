## 为什么要函数式编程
- React 就是
- vue 3 使用
- 可以抛弃 this
- 打包过程可以更好的利用 tree shaking 过滤无用的代码
- 方便测试 方便并行处理
- 有很多库可以帮助我们进行函数式编程 : lodash underscore ramda

## 概念
- Functional Programming , FP 是编程范式(面向过程, 面向对象)之一
- 用来描述数据之间的映射
```js
// 非函数式编程(面向过程)
let num1 = 1
let num2 = 2
let sum = num1 + num2 
console.log(sum)

// 函数式编程
function add (n1, n2) {
    return n1 + n2
}
let sum = add (1, 2)
console.log(sum)
```
