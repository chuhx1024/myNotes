## 升级 node
```sh
 npm i -g n

# 安装 最新版
$ n latest

# 切换
n

# 切换失效的解决办法

$ which node
# /usr/local/bin/node #举个例子


$ vi ~/.bash_profile
# insert 
export N_PREFIX=/usr/local/bin/node #node实际安装位置
export PATH=$N_PREFIX/bin:$PATH

# :wq保存退出

# 执行source使修改生效。

$ source ~/.bash_profile

```
